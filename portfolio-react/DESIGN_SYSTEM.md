## Design Aesthetic

### Style Characteristics
- **Clean & Modern**: Flat colors with distinct, often organic shapes
- **Paper Cutout Feel**: Layered shapes with organic, hand-cut edges
- **Abstract Geometry**: Playfully geometric and organic forms
- **Harmonious Palette**: Bright and muted tones working together

### Typography
- **Primary Font**: Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto'
- **Secondary Font**: Source Code Pro, 'Monaco', 'Consolas', monospace
- **Weight**: 600-700 for headers, 500 for body text

### Spacing System
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **xxl**: 48px

### Border Radius
- **sm**: 4px
- **md**: 8px
- **lg**: 12px
- **xl**: 16px
- **Paper Cutout**: Organic shapes using complex border-radius values

## Component Design

### Sticker/Paper Cutout Elements
- **Shape**: Irregular, organic border-radius for hand-cut paper effect
- **Shadows**: Multiple layered shadows for depth and dimension
- **Colors**: Vibrant Pantone colors with high contrast
- **Animation**: Bouncy, playful transitions with subtle rotation

### Grid Layout
- **Dynamic Sizing**: Images maintain original aspect ratios
- **Hover Effects**: Subtle scale and brightness adjustments
- **Centered Elements**: Perfect center alignment for stickers and overlays

## Usage Guidelines

### Color Application
1. **Primary Actions**: Use vibrant colors (Pantone 329 C, 1495 C, 805 C)
2. **Secondary Elements**: Use deeper tones (Pantone 526 C, 273 C, 3268 C)
3. **Accents**: Use contrasting colors for shapes and overlays
4. **Backgrounds**: Use lighter tones for main backgrounds

### Shape Design
1. **Organic Forms**: Use irregular border-radius for natural, hand-cut feel
2. **Layering**: Create depth with multiple shadow layers
3. **Contrast**: Ensure high contrast between shapes and backgrounds
4. **Animation**: Use playful, bouncy transitions for interactive elements

## Brand Personality
- **Playful**: Bouncy animations and organic shapes
- **Modern**: Clean typography and flat design principles
- **Artistic**: Hand-crafted paper cutout aesthetic
- **Professional**: High-quality color palette and consistent spacing
